## 🚀 How to Use

### To Run Locally
```bash
# 1. Add Firebase credentials to .env.local
#    (See GET_STARTED.md for instructions)

# 2. Start development server
npm run dev

# 3. Open browser
# http://localhost:3000
```

### To Deploy
```bash
# Option A: Vercel (Recommended)
vercel --prod

# Option B: Firebase Hosting
firebase deploy

# Option C: Self-hosted
npm run build && npm start
```
